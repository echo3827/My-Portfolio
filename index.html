Character Quiz

In this project I used conditional statements to create a Character Quiz based off the show "Avatar The Last Airbender". Within this website, the user is asked a series of questions which upon answering will gather information on their answers and draw a conclusion based off of that, then they will be presented with which character the website believes best fits their personality.
